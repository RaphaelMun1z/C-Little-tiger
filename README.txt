Instruções DOCKER:
1. Rode o container 'c-little-tiger';
2. Abra o terminal
3. Execute o seguinte comando: cd code/
4. Execute o seguinte comando para gerar executável: gcc -o app main.c interface/interface.c auth/auth.c db/db.c user/user.c games/bombs/bombs.c games/blaze/blaze.c games/roulette/roulette.c
5. Execute o seguinte comando para executar o programa: ./app
